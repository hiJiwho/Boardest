🔌 SDK & Manifest 가이드

웹 플러그인에서 사용할 수 있는 칠판 연동 API 목록입니다.

1. info.json / manifest.json 설정

{
  "id": "com.boardest.plugin",
  "name": "내 미니앱",
  "displayMode": "popup",
  "requiresCanvas": true,
  "role": "teacher"
}

2. window.boardest API

USB 장치 감지

const connected = await window.boardest.usbDetected();

칠판 설정 로드

const settings = await window.boardest.getSettings();

bst-Cloud 구글 드라이브 파일 탐색

const files = await window.boardest.getCloudFiles();
