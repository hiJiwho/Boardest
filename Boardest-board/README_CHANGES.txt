💡 주의사항
────────────────────────────────────────────────────────────────────────────

• USB 기능이 완전히 비활성화되었습니다.
  → 향후 활성화 필요 시 _checkUsbConnection() 주석 제거

• Windows는 EXE로 배포되며, MSI 인스톨러는 별도 도구 필요
  → WiX Toolset, NSIS 등 사용 가능

• 첫 빌드는 10-15분 소요 (이후 3-5분)
  → 의존성 다운로드 및 컴파일 시간

• Android 기기에 설치하려면 USB 디버깅 활성화 필요
  → adb install -r build/outputs/apk/app-release.apk
